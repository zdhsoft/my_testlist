## 📦 特性
1. 🍄 跨平台：一套代码，可以打包成windows版、Mac版、Linux版
2. 🌰 简单高效：只需学习 js 语言，同时支持vue、react、html等前端技术
3. 🌱 前端独立：理论上支持任何前端技术，编写出精美的UI效果
4. 🌴 工程化：可以用服务端的开发思维，来编写桌面软件
5. 🍁 高性能：事件驱动、非阻塞式IO
6. 🌷 功能丰富：前端、服务端的技术场景等
7. 💐 功能demo：桌面软件常见功能，后续逐步集成或提供demo
8. 🌻 更多功能请看文档
